# Santo Terço

Aplicativo web de oração do Santo Terço, convertido para React + TypeScript + Vite.

## Tecnologias

- React
- TypeScript
- Vite
- PWA-ready

## Scripts

```bash
npm install
npm run dev
npm run build
npm run preview
```

## Acesso

Acesse a versão publicada em:

[https://alan-nardin.github.io/santo-terco](https://alan-nardin.github.io/santo-terco)

## Funcionalidades

- seleção de mistérios por dia da semana
- guia de dezena com contador visual
- tema claro/escuro
- ajuste de zoom de texto
- layout responsivo e compatível com uso em dispositivos móveis
